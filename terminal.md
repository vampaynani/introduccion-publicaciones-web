# Comandos de terminal

- Listar directorio
```bash
ls
```
- Cambiar directorio
```bash
cd <nombre_del_directorio>
```
- Ir un nivel arriba del directorio
```bash
cd ..
```
- Ir a la carpeta de inicio
```bash
cd ~
```

